# Testes do Sitfor App

## Estrutura

- `unit/` - Testes unitários
- `e2e/` - Testes end-to-end
- `integration/` - Testes de integração
- `mocks/` - Mocks e fixtures

## Comandos

```bash
# Todos os testes
./bin/test

# Testes unitários
./bin/test unit

# Testes E2E
./bin/test e2e

# Testes com coverage
./bin/test coverage
```
